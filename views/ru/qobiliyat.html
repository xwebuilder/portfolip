Qobilatlarim

HTML

HTML — bu (ing. HyperText Markup Language, uzb. Giper matnli belgilash tili) bo’lib, umumjahon internet tarmog’idagi hujjatlar uchun standartlashtirilgan belgilash tili hisoblanadi.

CSS

CSS (ing. Cascading style sheets, uzb. Kaskadlangan stillar jadvali) — bu belgilash tilidan foydalanib yozilgan hujjatlarning ko’rinishini tasvirlash uchun ishlatiladigan rasmiy til hisoblanadi.

JavaScript

Javascript, bu Internet uchun katta boʻlmagan xaridor va server ilovalarni yaratishga moʻljallangan nisbatan oddiy jismga yoʻnaltirilgan til.

Bootstrap

Bootstrap HTML va CSS-ga asoslangan tipografiya, formalar, tugmalar, jadvallar, navigatsiya, modallar, rasm karusellari va boshqa ko’p narsalar uchun, shuningdek ixtiyoriy JavaScript plaginlarini o’z ichiga oladi.

GIT, BEM

Git bu bepul, ochiq manbali, Linus Torvalds tomonidan 2005 yilda yaratilgan versiyani boshqarish tizimi.
