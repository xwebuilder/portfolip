Qobilatlarim

HTML

HTML — bu (ing. HyperText Markup Language, uzb. Giper matnli belgilash tili) bo’lib, umumjahon internet tarmog’idagi hujjatlar uchun standartlashtirilgan belgilash tili hisoblanadi.

CSS

CSS (ing. Cascading style sheets, uzb. Kaskadlangan stillar jadvali) — bu belgilash tilidan foydalanib yozilgan hujjatlarning ko’rinishini tasvirlash uchun ishlatiladigan rasmiy til hisoblanadi.

JavaScript

Javascript, bu Internet uchun katta boʻlmagan xaridor va server ilovalarni yaratishga moʻljallangan nisbatan oddiy jismga yoʻnaltirilgan til.

Bootstrap

Bootstrap HTML va CSS-ga asoslangan tipografiya, formalar, tugmalar, jadvallar, navigatsiya, modallar, rasm karusellari va boshqa ko’p narsalar uchun, shuningdek ixtiyoriy JavaScript plaginlarini o’z ichiga oladi.

GIT, BEM

Git bu bepul, ochiq manbali, Linus Torvalds tomonidan 2005 yilda yaratilgan versiyani boshqarish tizimi.

jQuery

jQuery - bir JavaScript-kutubxona JavaScript, HTML va CSS o'zaro ajratilgan.

Gulp

Gulp veb-ishlab chiqish jarayonini tezlashtirish vositasidir. Gulp avtomatik ravishda eng tez-tez ishlatiladigan barcha vazifalarni bajaradi.

SASS

SASS (Sintaktik jihatdan ajoyib uslub jadvallari) - eng mashhurlaridan biri. Bu sizga tezroq va samaraliroq kodlashga imkon beruvchi CSS funktsiyalari to'plamidir.

LESS

LESS - bu CSS qo'shimchasi. LESS - bu programlanadigan CSS. LESS CSSni o'zgaruvchan, aralashma va operatsiyalar kabi dinamik qobiliyatlari bilan kengaytiradi.

PHP

PHP — skript tili boʻlib, web serverda oʻzgaruvchan HTML fayllarini yaratishda qoʻllaniladi. Hozirgi kunda deyarli barcha hosting-provayderlar mazkur tilni qoʻllab-quvvatlaydi.

WordPress

WordPress bu — kontentlarni boshqarish tizimi hisoblanib(CMS), PHP dasturlash tilida tuzilgan, qo’shimchasiga CSS stillari, JS kodlari, HTML teglari ham ishlatilgan. Bu CMS asosan blog saytlar tuzish uchun ishlatiladi, lekin hozirgi kunda istalgan murakkablikdagi saytlarni ham tuzish mumkin.
